# Pymental  [![PyPI version](https://badge.fury.io/py/pumental.svg)](https://badge.fury.io/py/pymental)

Pymental is a python client used for interacting with AWS Elemental Conductor

# Requirements

* Python3

# Installation

Install using pip:
```
pip install pymental
```
